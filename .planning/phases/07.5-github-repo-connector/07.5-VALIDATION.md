## Sampling Rate

- **After every task commit:** Run quick command (`-x` exit on first fail)
- **After every plan wave:** Run full suite command
- **Before `/gsd-verify-work`:** Full suite must be green AND manual GitHub smoke test green
- **Max feedback latency:** 25 seconds (quick), 90 seconds (full)
