**Apply to:** `backend/app/queue/tasks/scan_repo.py`.

### CC-6: Subprocess invocation — async, timeboxed, sanitized

The indexing job uses pure-Python work, so there's no existing subprocess analog in the backend. Closest pattern is `cli/infracanvas/main.py`'s sync subprocess use, but for async taskiq we need `asyncio.create_subprocess_exec` per RESEARCH § Taskiq Job. Treat this as a **new pattern unique to scan_repo.py** — copy the code skeleton from `07.5-RESEARCH.md` lines 411-508 verbatim. Key constraints carried from research:

- `env={**os.environ, "GIT_TERMINAL_PROMPT": "0"}` to suppress interactive auth prompts.
- `await asyncio.wait_for(proc.communicate(), timeout=N)` — kill + `await proc.wait()` on `TimeoutError`.
- **Sanitize stderr** before logging: `stderr.decode().replace(token, "***")`.
- `try/finally` around `tmp_root` with `shutil.rmtree(tmp_root, ignore_errors=True)`.
- Path-traversal guard: `(tmp_root / path).resolve()` startswith `tmp_root.resolve()`.
- Treat `infracanvas scan` rc=0 AND rc=1 as success (findings present is success); only rc=2 is a fatal scan error.

### CC-12: Dashboard — SSR data fetch via `backendFetch`

**Source:** `dashboard/lib/backend.ts:16-43`

```typescript
export async function backendFetch<T>(path: string, init?: RequestInit): Promise<T> {
  const { auth } = await import('@clerk/nextjs/server')
  const { getToken } = await auth()
  const token = await getToken()
  const backendUrl = process.env.BACKEND_URL
  const res = await fetch(`${backendUrl}${path}`, {
    ...init,
    headers: { Authorization: `Bearer ${token}`, 'Content-Type': 'application/json', ...init?.headers },
    cache: 'no-store',
  })
  if (!res.ok) throw new Error(`${res.status}`)
  return res.json() as Promise<T>

def mint_app_jwt() -> str:
    """5-minute App JWT, RS256. iat -60s clock-skew buffer."""
    private_key = load_pem_private_key(
        settings.github_app_private_key.encode(), password=None,
    )
    now = int(time.time())
    return jwt.encode(
        {"iat": now - 60, "exp": now + 5 * 60, "iss": settings.github_app_id},
        private_key,
        algorithm="RS256",
    )

async def mint_installation_token(installation_id: int) -> str:
    """Exchange App JWT for an installation access token."""
    app_jwt = mint_app_jwt()
    async with httpx.AsyncClient(timeout=10.0) as client:
        r = await client.post(
            f"https://api.github.com/app/installations/{installation_id}/access_tokens",
            headers={
                "Authorization": f"Bearer {app_jwt}",
                "Accept": "application/vnd.github+json",
                "X-GitHub-Api-Version": "2022-11-28",
            },
        )
        r.raise_for_status()
        return r.json()["token"]
```
