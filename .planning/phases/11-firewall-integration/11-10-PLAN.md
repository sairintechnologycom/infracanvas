Create agent/internal/fmc/client.go.

Package fmc. Imports: context, crypto/tls, encoding/json, errors, fmt, io, net/http, net/url, strconv, time, go.uber.org/zap, github.com/infracanvas/infracanvas/agent/internal/push.

Define ErrFMCAuth sentinel: var ErrFMCAuth = errors.New("fmc: authentication failed (non-retryable)").

Define type Client struct { http *http.Client; log *zap.Logger; token *fmcTokenInfo }.

Define NewClient(client *http.Client, log *zap.Logger) *Client. If nil, defaults like Plan 11-08.

Method baseURL(host, port int) string returns "https://" + host + ":" + strconv.Itoa(port). Default port 443.

Method acquireToken(ctx, host, port, user, pass): build POST request to baseURL + "/api/fmc_platform/v1/auth/generatetoken", set basic auth, execute. On non-2xx return ErrFMCAuth (401) or wrapped error. On 2xx: read response headers — c.token = &fmcTokenInfo{accessToken: resp.Header.Get("X-auth-access-token"), refreshToken: resp.Header.Get("X-auth-refresh-token"), domainUUID: resp.Header.Get("DOMAIN_UUID")}. Reset refreshCount to 0. Return nil.

Method refreshToken(ctx, host, port): if c.token.refreshCount >= 3, return ErrFMCAuth (per Pitfall 3 — must re-acquire from scratch). Build POST to /api/fmc_platform/v1/auth/refreshtoken with header "X-auth-access-token: " + accessToken AND "X-auth-refresh-token: " + refreshToken. On success, update c.token.accessToken from response header X-auth-access-token, increment refreshCount. On failure return ErrFMCAuth.

Method doGet(ctx, host, port, path string, out interface{}) error: build GET to baseURL+path, set "X-auth-access-token: " + c.token.accessToken header, execute.
- If 401: call refreshToken(ctx, host, port). If refresh succeeds, RETRY ONCE (rebuild req with new accessToken). If second 401 OR refresh failed: return ErrFMCAuth.
- If 2xx: io.ReadAll body, json.Unmarshal into out, return nil.
- Other status: return fmt.Errorf("fmc: get %s: status %d", path, resp.StatusCode).

Method paginatedGet(ctx, host, port, path string, accumulator func(rawItem json.RawMessage) error) error:
- Loop: doGet path → response has Items + Links.Next
- For each item in response Items: marshal back to json.RawMessage and call accumulator
- If Links.Next is non-empty, set path = Links.Next (extract path from URL via url.Parse), continue
- Else break and return nil
- Track page_count for logging

Method Pull(ctx, host, port, user, pass, siteID string) ([]push.FirewallRule, []push.FirewallNATRule, []push.FirewallObject, error):
1. err = c.acquireToken(ctx, host, port, user, pass) — return on err
2. List access policies: var policies fmcPolicyListResp; doGet("/api/fmc_config/v1/domain/" + c.token.domainUUID + "/policy/accesspolicies", &policies). If empty: return nil for rules (no policies = no rules). Else policyID = policies.Items[0].ID
3. Paginate access rules under policyID: var rules []push.FirewallRule. Use paginatedGet with path = "/api/fmc_config/v1/domain/{uuid}/policy/accesspolicies/" + policyID + "/accessrules?expanded=true&limit=100" — accumulator unmarshals each item into fmcAccessRule, normalizes to push.FirewallRule (action: map "ALLOW"→"permit", "BLOCK"→"deny"; src_cidr/dst_cidr from Literals[0].Value or Objects[0].Name; protocol/ports from DestinationPorts.Literals[0]; raw_blob = the full item JSON), append to rules
4. Same for NAT: list ftdnatpolicies → first → paginate manualnatrules → normalize each fmcNATRule into push.FirewallNATRule (interface_in=SourceInterface.Name, interface_out=DestinationInterface.Name, src_translation=TranslatedSource.Name, dst_translation=TranslatedDestination.Name, raw_blob=full item)
5. Same for objects: paginate /object/networks → normalize each fmcNetworkObject into push.FirewallObject (kind="host" if Type=="Host" else "network", name=Name, value=json.Marshal({"cidr": Value}), raw_blob=full item)
6. Log success: c.log.Info("fmc_pull_complete", host, page_count_total, len(rules), len(nats), len(objs))
7. Return rules, nats, objs, nil

CRITICAL — Pitfall 3 strict semantics: ONE refresh attempt per failed request. Second 401 returns ErrFMCAuth. The TestClient_TokenRefresh test (Wave 0) verifies this: httptest serves 401 → refresh succeeds → retry succeeds. A second test variant should serve 401 → refresh fails → ErrFMCAuth surfaces (Plan 11-01 Wave 0 stub may need to be extended; if so, do it here as part of green-ing the test).

CRITICAL — Pitfall 6: DOMAIN_UUID is REQUIRED on every fmc_config path. If c.token.domainUUID is empty after acquireToken, return fmt.Errorf("fmc: missing DOMAIN_UUID in auth response from %s", host) — observed in Pitfall 6 as a common misconfiguration.

CRITICAL — Pattern G: NEVER log accessToken, refreshToken, user, or pass. zap fields allowed: host, page_count, rules_count, nats_count, objects_count.

For TestClient_PaginatedAccessRules to GREEN: httptest serves multi-page response (page1 with Links.Next=page2 URL, page2 with no Next). Assertion: collected rules count = sum of both pages.
