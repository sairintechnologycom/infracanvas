Create agent/internal/asa/ssh_parser.go.

Package: asa. Imports: bufio, encoding/json, regexp, strings.

Compile regexes once at file scope using var blocks with regexp.MustCompile. Define these patterns:

1. aclLineRE: matches `access-list <name> extended (permit|deny) <protocol> <src-spec> <dst-spec>[ eq <port>][...]`
   Anchored, greedy-bounded. Example regex string (escape backslashes):
   `^access-list\s+(\S+)\s+extended\s+(permit|deny)\s+(\S+)\s+(host\s+\S+|any4?|any6?|\S+\s+\S+|object(?:-group)?\s+\S+)\s+(host\s+\S+|any4?|any6?|\S+\s+\S+|object(?:-group)?\s+\S+)(?:\s+eq\s+(\S+))?`

2. natLineRE: matches `nat (<iface_in>,<iface_out>) source (dynamic|static) <orig> <translated>[...]`
   Pattern: `^nat\s+\((\S+),(\S+)\)\s+source\s+(dynamic|static)\s+(\S+)\s+(\S+)(?:\s+destination\s+static\s+(\S+)\s+(\S+))?`

3. objectStartRE: matches `object network <NAME>` or `object service <NAME>`
   Pattern: `^object\s+(network|service)\s+(\S+)`

4. objectGroupStartRE: matches `object-group (network|service) <NAME>`
   Pattern: `^object-group\s+(network|service)\s+(\S+)`

5. objectHostRE: matches ` host <IP>` (leading space — these are continuation lines under an object block)
   Pattern: `^\s+host\s+(\S+)`

6. objectSubnetRE: matches ` subnet <IP> <MASK>`
   Pattern: `^\s+subnet\s+(\S+)\s+(\S+)`

7. networkObjectRE: matches ` network-object (host <IP>|<IP> <MASK>|object <NAME>)` for object-group members
   Pattern: `^\s+network-object\s+(host\s+\S+|\S+\s+\S+|object\s+\S+)`

State machine logic:
- `currentObject string` and `currentObjectKind string` track the active block
- A blank line, line not starting with whitespace, or new `object`/`object-group` definition closes the current block (flush accumulated value to objects slice, reset state)

For each line:
- Try aclLineRE.FindStringSubmatch. On match: build push.FirewallRule with src_cidr/dst_cidr from the address specs (resolve "host X.X.X.X" → "X.X.X.X/32", "any4"→"0.0.0.0/0", "object-group NAME"→"object:NAME" placeholder, "X.X.X.X Y.Y.Y.Y"→prefix from mask), action=match[2], protocol=match[3], ports=match[6] if present, position=incrementing-counter, raw_blob=json.RawMessage of `{"line": <original line as JSON-escaped string>}`. Append to rules slice.
- Else try natLineRE. On match: build push.FirewallNATRule with interface_in=match[1], interface_out=match[2], src_translation=match[5], dst_translation=match[7] if present, position=incrementing nat counter, raw_blob=line.
- Else if objectStartRE matches: flush prior object if any, set currentObject to NAME and currentObjectKind to "network" or "service".
- Else if objectGroupStartRE matches: flush prior object, set currentObject + kind="group".
- Else if currentObject != "" and a continuation line matches objectHostRE/objectSubnetRE/networkObjectRE: append to a temp value list.
- Else if line starts with non-whitespace and currentObject != "": flush.
- Otherwise: silently skip (T-10-05-03).

Helper functions:
- normalizeAddressSpec(s string) string — converts "host 1.2.3.4" → "1.2.3.4/32", "any" → "0.0.0.0/0", "1.2.3.4 255.255.255.0" → "1.2.3.4/24" (use net.IPMask conversion), "object NAME" → "object:NAME" or look up in objects slice if available (parser emits a placeholder; downstream join is Phase 12 territory).
- maskToBits(mask string) int — uses net.ParseIP + ones-count.

CRITICAL — T-10-05-03 from CAB packet: NO backreferences (\1, \2 in patterns are OK in submatch references but no `(?P=name)` or recursive patterns). NO unbounded quantifiers — every `.*` or `\S*` must be bounded by anchored position or replaced with `[^\s]+` or character classes.

CRITICAL — Pattern from ssh/parser.go: parser is a pure function. No I/O, no logging, no side effects. Errors are not returned — non-matching lines silently skipped.

For raw_blob: since each rule corresponds to a single config line, embed the line itself as a JSON object: `raw_blob = json.RawMessage(json.Marshal(map[string]string{"line": line}))`. The raw_blob field on push.FirewallRule is json.RawMessage so this is direct.
