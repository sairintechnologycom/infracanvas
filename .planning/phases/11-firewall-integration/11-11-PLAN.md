Create agent/internal/checkpoint/parser.go.

Package: checkpoint. Imports: encoding/json, fmt, github.com/infracanvas/infracanvas/agent/internal/push.

Define `func Parse(rulebaseJSON, natJSON, objectsJSON []byte) ([]push.FirewallRule, []push.FirewallNATRule, []push.FirewallObject, error)`:

1. var objResp ckpObjectsResp; if len(objectsJSON) > 0 { if err := json.Unmarshal(objectsJSON, &objResp); err != nil { return nil, nil, nil, fmt.Errorf("checkpoint: parse objects: %w", err) } }
2. Build objectsByUID := make(map[string]ckpObject); for _, o := range objResp.Objects { objectsByUID[o.UID] = o }
3. Also produce []push.FirewallObject by iterating objResp.Objects: kind = mapType(o.Type) (helper function), name = o.Name, value populated based on type (json.Marshal of an inline shape), raw_blob = json.RawMessage(json.Marshal(o)). Append to objects slice.
4. var rbResp ckpAccessRulebaseResp; same parse pattern. For each rule in rbResp.Rulebase: build push.FirewallRule with:
   - Position: rule.RuleNumber
   - Action: mapAction(rule.Action.Name) — "Accept"→"permit", "Drop"→"deny", "Reject"→"deny", default→"permit"
   - SrcCIDR: resolveAddrRefs(rule.Source, objectsByUID) — joins multiple refs with comma, or returns "any" if empty/Any
   - DstCIDR: resolveAddrRefs(rule.Destination, objectsByUID)
   - Protocol/Ports: resolveServiceRefs(rule.Service, objectsByUID) — picks first matching tcp/udp protocol, port from object's Port field
   - SrcZone/DstZone: empty (Checkpoint uses install-on, not zones)
   - RawBlob: json.RawMessage(json.Marshal(rule))
   Append to rules slice.
5. Same for NAT: var natResp ckpNATRulebaseResp; for each rule in natResp.Rulebase, build push.FirewallNATRule.
6. Return rules, nats, objects, nil

Helper functions:
- mapType(ckpType string) string: "host"→"host", "network"→"network", strings starting with "service-"→"service", "group"|"service-group"→"group"
- mapAction(actName string) string: "Accept"→"permit", "Drop"|"Reject"→"deny", default→strings.ToLower(actName)
- resolveAddrRefs(refs []ckpRef, byUID map[string]ckpObject) string: for each ref, if obj, ok := byUID[ref.UID]; ok then if obj.IPv4Address != "" return obj.IPv4Address+"/32", else if obj.Subnet4 != "" return obj.Subnet4+"/"+strconv.Itoa(obj.MaskLength4); else fall back to ref.Name. Join multiple with ",". Empty → "any" (translates to 0.0.0.0/0 at consumer).
- resolveServiceRefs(refs []ckpRef, byUID): pick first; return obj.Type (tcp/udp/icmp) + port info if available.
