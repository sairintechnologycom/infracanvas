Define NewRESTCollector(client *http.Client, log *zap.Logger) *RESTCollector. If client is nil, default to &http.Client{Timeout: 10 * time.Second, Transport: &http.Transport{TLSClientConfig: &tls.Config{InsecureSkipVerify: false}}}. (The InsecureSkipVerify decision is CAB-documented in Plan 11-13 — false by default, operator can flip via future config knob.)

Define ErrASAAuth as a sentinel error: var ErrASAAuth = errors.New("asa-rest: authentication failed (non-retryable)").

Method baseURL(host string, port int) string returns "https://" + host + ":" + strconv.Itoa(port). Default port: if port==0, use 443.

Method acquireToken(ctx, host, port, user, pass) (string, error):
1. Build POST request to baseURL + "/api/tokenservices" with empty body
2. Set basic auth via req.SetBasicAuth(user, pass) — never log this
3. Execute http.Do
4. If status == 401: return "", ErrASAAuth
5. If status not 2xx: return "", fmt.Errorf("asa-rest: token %s: status %d", host, resp.StatusCode)
6. Read X-Auth-Token header from response — return value or error if empty

Method deleteToken(ctx, host, port, tok) just calls DELETE baseURL + "/api/tokenservices/" + url.PathEscape(tok), ignores response, logs WARN on err with host (no tok in log).

Method Pull(ctx, host, port, user, pass, siteID string) ([]push.FirewallRule, []push.FirewallNATRule, []push.FirewallObject, error):
1. tok, err = acquireToken — return early on err
2. defer c.deleteToken(ctx, host, port, tok)
3. Helper closure doGet(path string, out interface{}) error: build GET request, set "X-Auth-Token: " + tok header, execute, unmarshal JSON into out
4. var aclResp asaACLResponse — call doGet("/api/access/in/outside/rules", &aclResp). Wrap any error as fmt.Errorf("asa-rest: acl %s: %w", host, err)
5. var natResp asaNATResponse — call doGet("/api/nat", &natResp)
6. var objResp asaObjectsResponse — call doGet("/api/objects/networkobjects", &objResp)
7. Normalize each into push.FirewallRule / push.FirewallNATRule / push.FirewallObject. The action field maps Permit==true to "permit" and false to "deny". The raw_blob captures the original item by re-marshalling it with json.Marshal (assign to RawMessage and store in raw_blob field of push struct).
8. Return the three slices, nil
9. Log INFO at end: c.log.Info("asa_rest_pull_complete", zap.String("host", host), zap.Int("rules", len(rules)), zap.Int("nats", len(nats)), zap.Int("objects", len(objs)))
